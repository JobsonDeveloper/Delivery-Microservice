# 🛍️ Delivery-Microservice

### 📌 About This Project

Microservice focused on the creation and management of deliveries.
This project is one of five microservices that make up an e-commerce system.

User Microservice: https://github.com/JobsonDeveloper/User-Microservice

Product Microservice: https://github.com/JobsonDeveloper/Product-Microservice

Sale Microservice: https://github.com/JobsonDeveloper/Sale-Microservice

Payment Microservice: https://github.com/JobsonDeveloper/Payment-Microservice

Delivery Microservice: https://github.com/JobsonDeveloper/Delivery-Microservice

### ⚙️ Basic Configuration
1️⃣ Create a Docker network for all microservices to communicate:
```
docker network create ecommerce-network
```

2️⃣ Start the Kafka and the MongoDB in the Docker

Link to clone it: https://github.com/JobsonDeveloper/E-Sale

3️⃣ Build and start the project with Docker:

```
docker compose up --build
```
